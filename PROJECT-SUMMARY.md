## 🚀 Getting Started (3 Steps)

### Step 1: Install Dependencies
```bash
cd ismail-portfolio
npm install
```

### Step 2: Run Development Server
```bash
npm run dev
```

### Step 3: Open Browser
Visit: **http://localhost:3000**

You'll see your portfolio with placeholder images!

## 📸 Adding Your Photos

Replace placeholder images in `/public/images/` with your actual photos.

**Required Images:**
- `hero-1.jpg` & `hero-2.jpg` - Homepage featured images
- `gallery-1.jpg` to `gallery-6.jpg` - Gallery page
- `commission-1.jpg` & `commission-2.jpg` - Commission page
- `personal-1.jpg` to `personal-4.jpg` - Personal works
- `about-portrait.jpg` - Your portrait photo
- `book-cover.jpg` - Book cover
- `book-preview-1.jpg` to `book-preview-3.jpg` - Book interior shots

## 📝 Customizing Content

### Update Your Information:
1. **About Page Bio:** Edit `app/about/page.tsx`
2. **Contact Email:** Edit `app/contact/page.tsx` (search for "hello@")
3. **Social Links:** Edit `app/contact/page.tsx`
4. **Book Details:** Edit `app/book/page.tsx`

All content is in simple TypeScript/React files - easy to edit!

## ✏️ Easy Updates

**To add more gallery images:**
1. Add image to `/public/images/`
2. Edit `app/gallery/page.tsx`
3. Add entry to `galleryImages` array
4. Save and redeploy

**To change colors/fonts:**
- Edit `app/globals.css`
- All styling uses CSS variables for easy theming
